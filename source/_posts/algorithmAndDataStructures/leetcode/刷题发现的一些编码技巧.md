刷题发现的一些编码技巧：



1.判断是否为偶数，奇数

不要再用 n % 2 == 0 是偶数， n % 2  != 0是奇数了。位运算可以更快，也更加优雅，

n & 1 == 1 就是奇数 n & 1 == 0 是偶数



2.char数组里面的元素替换成它的前一个字符

之前我都是这么写的：arr[j] = (char) (arr[j] - 1);

看见别人写的题解之后才发现可以这么写：arr[j]--

代码优雅很多，也更加简洁
