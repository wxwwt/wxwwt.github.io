mybatis流式查询

发表于2023-11-05|更新于2024-08-15
|阅读量:

导语：

有些时候我们所需要查询的数据量比较大，但是jvm内存又是有限制的，数据量过大会导致内存溢出。这个时候就可以使用流式查询，
数据一条条的返回，处理完一条在拿下一条数据，这样每次在内存里面的数据其实很小，不会导致内存溢出。
本文里面会讲到jdbc的流式查询和mybatis的流式查询。
